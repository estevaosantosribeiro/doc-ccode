Introdução ao Ccode

Bem-vindo à documentação do Ccode, uma poderosa ferramenta de desenvolvimento que permite a criação de jogos e aplicações interativas de forma simples e intuitiva. O Ccode foi projetado para atender tanto desenvolvedores iniciantes quanto experientes, oferecendo uma interface amigável e recursos avançados que facilitam a implementação de funcionalidades complexas.

Criar Objeto

Primeiro, entre no script que criou. Então, clique no ícone de "+" e depois vá em "Objetos".

Parâmetro "Nome do Objeto": Serve para adicionar um identificador ao objeto criado, para que os outros blocos possam reconhecê-lo.

Parâmetro "Width": Significa largura. Parâmetro "Height": Significa altura. O próprio nome já diz, serve para alterar a largura e altura do objeto.

Parâmetro "Position X" e "Y": Significa posição X e Y. O próprio nome já diz, mostra a posição do objeto.

Criar Objeto com Partes de Sprites

Na mesma tela de objetos, há esse bloco como a segunda opção. Esse bloco é utilizado para criação de animações. Ele recorta uma spritesheet em retângulos.

Definir Posição do Objeto
  • Define a posição do objeto.
  • Parâmetro "Nome do Objeto": Serve para colocar o nome do objeto que deseja definir a posição.
  • Parâmetro "Position X" e "Y": Serve para definir a posição do objeto especificado no parâmetro "Nome do Objeto".
Definir Posição X do Objeto
  • Parâmetro "Nome do Objeto": Serve para especificar o nome do objeto que deseja definir a posição.
  • Parâmetro "Posição X": Serve para definir a posição X do objeto especificado no parâmetro "Nome do Objeto".
Definir Posição Y do Objeto
  • Parâmetro "Nome do Objeto": Serve para especificar o nome do objeto que deseja definir a posição.
  • Parâmetro "Posição Y": Serve para definir a posição Y do objeto especificado no parâmetro "Nome do Objeto".
Definir Tamanho do Objeto
  • Parâmetro "Nome do Objeto": Serve para especificar o nome do objeto que deseja definir o tamanho.
  • Parâmetro "Size": Significa tamanho. Serve para manipular o tamanho do objeto especificado no parâmetro "Nome do Objeto".
Definir Rotação do Objeto
  • Parâmetro "Nome do Objeto": Serve para especificar o objeto que deseja manipular a rotação.
  • Parâmetro "Rotação": Serve para manipular a rotação do objeto.
Parâmetros de Objeto
Nome do Objeto:

Aqui seria o nome do seu objeto, para que os outros blocos possam identificar o objeto criado. É importante nomear os objetos.

Posição X e Posição Y

Parâmetro responsável por permitir manipular a posição X e Y de textos, objetos, etc.

Largura e Altura

Parâmetro "Largura" e "Altura" serve para manipular a largura e a altura de objetos, textos, etc.

Rotação

Serve para manipular a rotação do objeto.

RGB

Serve para definir a cor do objeto.

Parâmetros de Texto
Texto

Parâmetro "Texto" que normalmente se encontra em blocos que criam textos. Ele define o texto que será mostrado na sua tela.

Nome

Serve como identificador para que os outros blocos possam manipular o texto que você criou.

RGB

O parâmetro "RGB" serve para definir uma cor ao texto.

Variáveis
Evento

Variáveis do tipo evento só pode ser utilizada apenas dentro daquele conjunto de blocos que ela foi definida. Ou seja, se você definir essa variável do tipo evento no bloco de quando iniciar e depois tentar mencioná-la no bloco de quando tocado, Não será possível pois essa variável foi definida como tipo evento assim sendo ela privada somente no bloco que foi definida.

Como você está acessando no editor de expressões, aquele estará todas as variáveis que você já criou do tipo evento.

script

variaveis que ficam no "script", significa que você só pode acessar aquela variavel apenas neste script que você criou

Projeto

Quando existe variaveis no "projeto", a variavel se torna acessivel no projeto inteiro, você pode acessar a variavel em qual quer pasta de script
